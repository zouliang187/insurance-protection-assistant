# 保障先知：公益保险配置助手

一个面向中国普通家庭的公益保险科普与保障配置项目。

项目初心不是卖保险，而是帮助普通人理解风险、识别保障缺口、看懂保险条款，并在合规边界内做出更理性的保障选择。第一版聚焦“年轻人为父母补基础保障”的真实场景。

在线原型：[zouliang187.github.io/insurance-protection-assistant](https://zouliang187.github.io/insurance-protection-assistant/)

> 重要说明：本项目仅用于保险科普、信息整理和保障路径参考，不构成保险销售、投保建议、个性化保险咨询或理赔承诺。具体投保、承保和理赔结果应以保险公司官方条款、健康告知、核保结论和监管要求为准。

## 项目定位

> 面向普通家庭的公益保险教育与保障配置助手，帮助用户先理解风险缺口，再用透明规则筛选基础保障型产品。

第一阶段优先服务：

- 年轻人为父母配置基础保障
- 普通家庭配置第一份保障
- 中低收入家庭用有限预算降低灾难性风险敞口

第一阶段优先聚焦：

- 基本医保理解
- 百万医疗险
- 惠民保
- 意外险
- 定期寿险
- 一年期重疾险或防癌相关保障

## 合规边界

本项目当前定位为保险科普、条款理解和保障路径参考工具，不提供保险购买链接，不做佣金导向产品排名，不承诺投保或理赔结果。

为了降低误用风险，项目维护会遵循以下原则：

- 不输出“必须买某款产品”的结论。
- 不根据佣金、返利或商业合作决定展示顺序。
- 不鼓励用户隐瞒健康状况或绕过健康告知。
- 所有关键资料应尽量引用监管、医保、保险公司条款等可复核来源。
- 涉及具体产品条款时，应记录资料更新时间和来源链接。
