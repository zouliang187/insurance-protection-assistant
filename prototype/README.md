# 网站原型 v0.1

这是父母保险选择器的第一版静态原型。

在线访问：

- `https://zouliang187.github.io/insurance-protection-assistant/`

## 打开方式

可以直接打开：

- `prototype/index.html`

也可以在 `prototype` 目录下启动本地预览服务，然后访问：

- `http://localhost:4173`

## 当前包含

- 首屏父母保险选择器问卷
- 动态保障路径结果
- 投保前核对清单
- 产品字段对比示例
- 推荐阅读入口
- 公益与合规边界说明

## 当前不包含

- 后端服务
- 用户账号
- 真实产品数据库
- 购买链接
- AI 条款解析

## 合规提示

本原型只做保险科普、保障路径参考和投保前核对清单，不构成保险销售、投保建议、个性化保险咨询或理赔承诺。具体结果以保险公司官方条款、健康告知、核保和理赔结论为准。
